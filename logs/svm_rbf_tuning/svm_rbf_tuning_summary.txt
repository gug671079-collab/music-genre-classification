SVM-RBF GroupKFold 调参实验
==================================================

一、实验目的
在与 KNN 调参相同的数据处理、分组方式、交叉验证方式和评价指标下，对 SVM-RBF 进行参数搜索。
本实验使用完整数据集，不额外划分 final_test，主要评价指标为 Macro-F1。

二、数据路径
读取数据：data/processed/processed_features_3_sec_V2.csv

三、数据规模
样本数量：9857
track_id 数量：987

四、类别分布
label

classical     998
country       997
disco         989
hiphop        978
jazz         1000
metal         930
pop           980
reggae        987
rock          998

五、字段处理
目标变量：label
分组字段：track_id
删除的非模型字段：
filename, genre_from_filename, track_id, segment_id, length

六、模型与验证方式
使用的模型：SVC(kernel="rbf")
验证方式：GroupKFold(n_splits=5)
主要评价指标：Macro-F1

七、参数搜索范围
{'model__C': [0.1, 1, 10, 100], 'model__gamma': ['scale', 0.001, 0.01, 0.1, 1]}
C 控制正则化强度；gamma 控制 RBF 核函数中样本影响范围。

八、最优参数与分数
最优参数：{'model__C': 10, 'model__gamma': 'scale'}
最优 mean_test_f1_macro：0.7243
对应 std_test_f1_macro：0.0200
对应 mean_test_accuracy：0.7268
对应 mean_test_f1_weighted：0.7238

十四、与 Baseline V3 和 KNN 调参的比较说明
本实验与 Baseline V3、KNN 调参实验保持相同的数据输入、字段处理方式、标准化流程、GroupKFold 分组方式和主要评价指标，因此可以用于后续模型横向比较。Baseline V3 是固定参数 KNN，本实验是在相同评估框架下对 SVM-RBF 的 C 和 gamma 进行参数搜索。
